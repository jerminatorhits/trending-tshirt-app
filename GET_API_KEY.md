## Detailed Instructions

### Step 1: Visit Stability AI
Open your browser and go to: **https://platform.stability.ai/**

### Step 2: Sign Up
- Click the **"Sign Up"** button (usually in the top right)
- You can sign up with:
  - Email and password
  - Google account
  - GitHub account

### Step 3: Verify Email
- Check your email inbox
- Click the verification link
- You'll be redirected back to the platform

### Step 4: Navigate to API Keys
Once logged in:
- Click on your **profile icon** (top right)
- Select **"API Keys"** from the dropdown
- OR go directly to: https://platform.stability.ai/account/keys

### Step 5: Create Your API Key
- Click the **"Create API Key"** or **"Generate API Key"** button
- Give it a name (e.g., "T-Shirt Generator App")
- Click **"Create"** or **"Generate"**

### Step 6: Copy Your Key
- **IMPORTANT**: Copy the key immediately - you won't be able to see it again!
- The key will look like: `sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
- Make sure you copy the ENTIRE key

### Step 7: Add to Your Project
Once you have the key, I can help you add it to your `.env` file!
